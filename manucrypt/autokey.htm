Extended Vigenère autokey cypher ("Autokey")

Introduction

Caesar's cypher uses a set shift that all of the plaintext is shifted modulo 26 (to wrap around the alphabet).

Simple Vigenère autokey cypher used a primer key of a single letter. Instead of applying same shift to everything, the shift is initially determined by the primer letter (its numeric value). Then autokey means that the original plaintext is then used to encode subsequent characters. Decoding requires using the result after the first letter.

"Autokey" uses a primer of multiple letters, also prepended to the plaintext as key. Decoding requires using the result after running out of the provided primer.

Practical weakening simplification: move away from base 26 to base 10, redefining the alphabet to the recipient alongside each message.
